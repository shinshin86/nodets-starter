# nodets-starter

My starter kit of Node.js script using TypeScript

## Commands

Scripts start

```bash
yarn start
```

Scrpts Build (Production)

```bash
yarn build
```

Development (Build and Start with development mode)

```bash
yarn dev
```

Lint and Preitter

```bash
yarn fmt
```

Link check (Not fix)

```bash
yarn lint:check
```

Prettier check (Not write)

```bash
yarn prettier:check
```
